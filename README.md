# ШинМаркет — Прототип мобильного приложения

Интерактивный UI-прототип приложения для покупки и продажи шин на рынке Казахстана.

## 🔗 Демо

https://zzkhxyz.github.io/shinmarket/

## 📱 Экраны

- **Онбординг** — выбор языка (Рус / Қаз), приветствие
- **Главная** — баннер акций, категории, популярные товары
- **Каталог** — поиск, фильтры по типу/сезону/городу, список товаров
- **Карточка товара** — галерея, цена, рассрочка Kaspi, продавец, отзывы
- **Продать** — форма подачи объявления с фото и параметрами шины
- **Оплата** — выбор способа оплаты (Kaspi Pay, наличные, карта)
- **Отзывы** — рейтинг с барами, карточки покупателей
- **История заказов** — статусы доставки, повторная покупка
- **Профиль** — настройки языка, доступность, уведомления

## 🎯 UX-принципы

- Нижняя таб-навигация по паттерну Kaspi / Kolesa
- Двуязычный интерфейс: русский и казахский
- Доступность для аудитории 14–65 лет (крупный шрифт, высокий контраст)
- Интеграция с Kaspi Pay и рассрочкой 0-0-24
- Верификация продавцов и отзывы только от покупателей

## 🛠 Технологии

- Чистый HTML / CSS / JavaScript — без фреймворков
- Tabler Icons
- Адаптирован под мобильный браузер (viewport, safe-area, iOS/Android)

## 📋 Контекст
Основан на анализе UX-паттернов казахстанских приложений: Kaspi.kz, Kolesa.kz, Krisha.kz, Wildberries.
